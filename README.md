# Friends & Builders: A GitHub Adventure with Mona, Ducky, and Copilot

_Join your favorite characters on a three-level GitHub adventure — branch with Mona, deploy with Ducky, and forge code with Copilot!_

<img src="https://github.com/user-attachments/assets/631758ce-770a-49c3-8dfc-449f01b58a33" alt="Mona, Ducky, and Copilot — Friends & Builders" width="600" />

## Welcome

- **Who is this for**: GitHub beginners and intermediate users who want to learn core GitHub workflows through fun, character-driven challenges.
- **What you'll learn**: Branching & commits, GitHub Actions CI/CD, and GitHub Copilot AI-assisted development.
- **What you'll build**: A branch, a CI workflow, and Copilot-assisted Python utilities — all in one epic adventure!
- **Prerequisites**:
  - A GitHub account
  - Basic familiarity with Markdown (helpful but not required)

- **How long**: This exercise takes less than 45 minutes to complete.

<img src="https://github.com/user-attachments/assets/94c61bed-f731-4bc3-b63f-ad6772379848" alt="Mona's Dev Den, Ducky's Deployment Dock, and Copilot's AI Forge" width="600" />

In this exercise, you will:

1. 🐱‍💻 **Enter Mona's Dev Den** — Create a branch, commit a file, and open a Pull Request.
1. 🐥 **Visit Ducky's Deployment Dock** — Build a GitHub Actions CI workflow and merge your code.
1. ⚡ **Forge at Copilot's AI Forge** — Co-create Python utilities with GitHub Copilot and generate tests with Copilot Chat.

### How to start this exercise

Simply copy the exercise to your account, then give your favorite Octocat (Mona) **about 20 seconds** to prepare the first lesson, then **refresh the page**.
